Ph.D. projects

Behavior measurement using wearable sensors

My projects at imec were focused on developing preventative mHealth and self-care solutions via passive monitoring of daily behaviors (e.g. smoking).

  1. Firstly, I engineered a non-invasive monitoring system to monitor the moment-to-moment biopsychosocialenvironmental context of each user. The system was comprised of three wearable sensors, a cloud database, a website and a mobile phone app.

  2. Secondly, by fusing these data collected from all the sensors, we build up a model that was able to understand and adapt to user's phenotype and varying context.

  3. Finally, we could give dynamically personalized and temporally-precise suggestion/interventions to each individual, so that we can nudge people with unhealthy lifestyle to be healthy and stay healthy.
Publication
  • Zhai, Donghui, et al. "Reveal Temporal Patterns of Smoking Behavior in Real Life Using Data Acquired through Automatic Tracking Systems." 2020 42nd Annual International Conference of the IEEE Engineering in Medicine & Biology Society (EMBC). IEEE, 2020.
  • Zhai, Donghui, et al. "Ambulatory Smoking Habits Investigation based on Physiology and Context (ASSIST) using wearable sensors and mobile phones: protocol for an observational study." BMJ open 9.9 (2019): e028284.
